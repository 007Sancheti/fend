## Repo

Inside the repo you'll find the following folders used to organize resources for FEND. Each lesson folder is composed of **assets**, **concepts**, **keynotes**, and an optional **problem set**.

```
fend/
└── lessons/
     ├── L0
     ├── L1
     ├── L2
     │    ├── assets
     │    ├── concepts
     │    ├── keynotes
     │    └── problem-set
     ├── L3
     ├── L4
     └── ...
```
